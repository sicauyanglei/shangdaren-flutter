# 上大人字牌 - 胡牌规则

## 1. 胡牌方式

胡牌分为两种方式：
- **点炮**: 胡别人打出的牌，使用基础倍数
- **自摸**: 自己摸到胡牌，在点炮倍数基础上+1倍

## 2. 普通胡牌（按胡数区分）

**基本条件**：胡牌时手牌除去句子/坎，只能有一个对子或者一个半靠

| 类型 | 胡数要求 | 点炮倍数 | 自摸倍数 | 说明 |
|------|----------|----------|----------|------|
| 普通胡 | 12~21胡 | 0倍 | 1倍 | 标准胡牌 |
| 卡胡 | 11胡 | 1倍 | 2倍 | 最低胡牌门槛 |
| 台卡 | 22胡 | 2倍 | 3倍 | 台数+卡胡 |
| 台胡 | 23~32胡 | 1倍 | 2倍 | 达到台数要求 |
| 重台卡 | 33胡 | 7倍 | 8倍 | 重台+卡胡 |
| 重台胡 | ≥34胡 | 6倍 | 7倍 | 最高普通胡 |

## 3. 特殊胡牌类型

### 3.1 枯胡
- **条件**: 手牌全部是坎（三张相同）外加一对，没有吃过句子
- **特点**: 纯粹靠坎组合胡牌
- **点炮倍数**: 5倍
- **自摸倍数**: 6倍

### 3.2 清枯胡
- **条件**: 枯胡 + 组合牌和手牌不能有上/福
- **特点**: 手牌和组合牌中没有上/福字
- **点炮倍数**: 6倍
- **自摸倍数**: 7倍

### 3.3 清枯台卡
- **条件**: 清枯 + 台卡（22胡）
- **特点**: 清枯与台卡的组合
- **点炮倍数**: 8倍
- **自摸倍数**: 9倍

### 3.4 黑元
- **条件**: 
  - 手牌除了一个半靠，其它全是句子
  - 只有吃的牌（没有碰牌）
  - 手牌和吃的牌没有"上大人"和"福禄寿"任何一个字
- **特点**: 排除特殊句子的纯句子组合
- **点炮倍数**: 4倍
- **自摸倍数**: 5倍

### 3.5 红元
- **条件**: 
  - 手牌除了一个半靠，其它全是句子
  - 只有吃的牌（没有碰牌）
  - 手牌和吃的牌必须保证上/福字数量≥3个
- **精数细分**: 
  | 类型 | 上/福数量 | 点炮倍数 | 自摸倍数 | 
  |------|-----------|----------|----------| 
  | 红元3精 | 3个 | 3倍 | 4倍 | 
  | 红元4精 | 4个 | 4倍 | 5倍 | 
  | 红元5精 | 5个 | 5倍 | 6倍 | 
  | 红元6精 | 6个 | 6倍 | 7倍 | 
  | 红元7精 | 7个 | 7倍 | 8倍 | 

### 3.6 十对
- **条件**: 手牌由10个对子组成
- **特殊规则**: 4张相同的字算两对（如上上上上算两对）
- **特点**: 特殊的对子胡牌方式
- **点炮倍数**: 10倍
- **自摸倍数**: 11倍
- **示例**: 
  - 普通十对：上上 + 大大 + 人人 + 丘丘 + 乙乙 + 己己 + 化化 + 三三 + 千千 + 七七 
  - 含4张相同：上上上上 + 大大 + 人人 + 丘丘 + 乙乙 + 己己 + 化化 + 三三 + 千千（上上上上算两对，共10对） 

## 4. 组合胡牌类型

| 类型 | 条件 | 点炮倍数 | 自摸倍数 | 说明 | 
|------|------|----------|----------|------| 
| 清胡 | 无上/福 + 无上大人/福禄寿半靠 + 12~21胡 | 1倍 | 2倍 | 手牌和组合牌中没有上/福，没有上大人或福禄寿的半靠，总胡数12~21胡 | 
| 清卡胡 | 无上/福 + 无上大人/福禄寿半靠 + 11胡 | 2倍 | 3倍 | 手牌和组合牌中没有上/福，没有上大人或福禄寿的半靠，总胡数11胡 | 
| 枯重台卡 | 枯胡 + 重台卡(33胡) | 12倍 | 13倍 | 组合条件 |

## 6. 胡数计算规则

### 6.1 单张牌胡数
| 牌型 | 数量 | 胡数 |
|------|------|------|
| **上、福**（特殊牌） | 1张 | 4胡 |
| **上、福**（特殊牌） | 2张 | 8胡 |
| **上、福**（特殊牌） | 3张及以上 | 12胡 |
| **其他牌** | 3张及以上 | 3胡 |

### 6.2 句子胡数
| 条件 | 胡数 |
|------|------|
| 同一句有两张不同的牌，且其中一张是"上"或"福" | 4胡 |

### 6.3 组合牌胡数
| 操作 | 胡数 |
|------|------|
| 吃 | 0胡 |
| 碰 | 2胡 |
| 招 | 6胡 |

## 7. 计分公式

```
基础分 = 5分
倍数基数 = 2分
飘分 = 玩家选择的飘分（0、5、10、20）

分数 = 基础分 + 倍数 × 倍数基数 + 飘分
自摸时：分数 × 2
```

## 8. 胡牌条件

1. **最低胡数**：11胡
2. **听牌状态**：玩家必须先听牌才能胡牌
3. **自摸**：摸牌后手牌满足胡牌条件
4. **点炮**：其他玩家出牌后，手牌+该牌满足胡牌条件

## 9. 牌色分类

- **红色牌**：上、丘、化、七、尔、八、佳、福
- **绿色牌**：大、乙、三、十、小、九、作、禄
- **黑色牌**：人、己、千、土、生、子、亡、寿

## 10. 句子分类

- **第1句**：上、大、人
- **第2句**：丘、乙、己
- **第3句**：化、三、千
- **第4句**：七、十、土
- **第5句**：尔、小、生
- **第6句**：八、九、子
- **第7句**：佳、作、亡
- **第8句**：福、禄、寿

## 11. 特殊句子

- **上大人**：第1句（上、大、人）
- **福禄寿**：第8句（福、禄、寿）
