## 组合牌型

| 牌型 | 组成            | 胡数      |
| -- | ------------- | ------- |
| 普靠  | 同组不同2张(不含上/福) | 0       |
| 银靠 | 大人或禄寿         | 0       |
| 精靠 | 同组不同2张，含上/福   | 4       |
| 普单  | 单张(非上/福)      | 0       |
| 银单  | 单张(大/人/禄/寿)      | 0       |
| 精单 | 上或福           | 4       |
| 普对  | 同字2张(非上/福)    | 0       |
| 银对  | 同字2张(大/人/禄/寿)    | 0       |
| 金对 | 上上或福福         | 8       |
| 普坎  | 同字3张(非上/福)    | 组合2/手牌3（炮普坎2） |
| 精坎 | 上上上或福福福       | 12      |
| 普招  | 同字4张(非上/福)    | 6       |
| 精招 | 上上上上或福福福福     | 16      ||
| 普句  | 同组不同3张(组2-7)  | 0       |
| 精句 | 同组不同3张(组1/8)  | 4       |
