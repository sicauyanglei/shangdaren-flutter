
## 分数计算
记作某种胡牌类型的基础倍数即点炮的倍数是B
| 角色     | 公式                                     |
| ------ | -------------------------------------- |
| 输家(点炮)     | 底分 + B × 倍数基数 + 飘分(己+赢家)               |
| 赢家(点炮) | 底分 + B × 倍数基数 + 飘分(己+点炮者)              |
| 赢家(自摸) | 2 × (底分 + (B+1) × 倍数基数) + 飘分(己×2 + 他家) |
| 输家(自摸) | 底分 + (B+1) × 倍数基数 + 飘分(己+赢家)               |
