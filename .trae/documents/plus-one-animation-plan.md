# 摸牌+1动画效果设计方案

## 场景说明

人类玩家摸牌/招后补摸牌时，如果手牌中已有同字卡牌，右上角数字徽章需要显示"+1"动画效果。

当前徽章样式：红色圆形（半径18px），白色数字，位于卡牌右上角。

## 方案列表

| 方案 | 名称 | 描述 |
|------|------|------|
| A | +1飞入徽章 | 金色"+1"从右上方飞入徽章位置，短暂停留后上浮消散 |
| B | +1弹出缩放 | 绿色"+1"从徽章位置弹出缩放，弹性效果后缩小消散 |
| C | 徽章数字跳动 | 徽章本身弹性缩放跳动 + 白色闪光圈扩散 |
| D | +1上浮消散 | 橙色"+1"从徽章位置缓慢上浮，逐渐变淡消散 |
| E | 徽章脉冲放大+光圈 | 徽章整体放大脉冲 + 红色光圈扩散，无需额外文字 |
| F | +1从徽章弹出+徽章跳动 | 红色"+1"从徽章位置向上弹出消散，同时徽章弹性跳动 |

## 实现步骤

1. 创建 HTML 文件展示 6 种方案
2. 在浏览器中预览
3. 用户选择后在 Flutter 代码中实现
