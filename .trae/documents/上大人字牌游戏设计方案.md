# 上大人字牌游戏设计方案

## 一、项目概述

设计一个完整的"上大人"字牌游戏，包含24种字牌（每种4张，共96张），支持完整的胡数计算和多种胡牌类型判定。游戏为3人对战模式，每组由8局游戏组成。

### 1.1 玩家角色定义
- **玩家1**: AI玩家
- **我**: 人类玩家
- **玩家2**: AI玩家

### 1.2 运行平台要求
游戏需要支持以下平台：
- **Android** - 安卓手机和平板
- **iOS** - 苹果手机和平板
- **Windows** - Windows电脑

### 1.3 技术选型
为支持跨平台运行，推荐使用以下技术方案：
- **React Native** - 支持Android和iOS原生应用
- **Electron** - 支持Windows桌面应用
- **React** - Web版本，可打包为各平台应用
- **TypeScript** - 统一的代码语言

### 1.4 代码共享策略
```
shangdaren-game/
├── core/                    # 核心游戏逻辑（跨平台共享）
│   ├── types/              # 类型定义

```

## 二、字牌定义

### 2.1 字牌组成（8句话，24个字）

| 第二句 | 丘(红)、乙(绿)、己(黑) | 丘乙己 |
| 第三句 | 化(红)、三(绿)、千(黑) | 化三千 |
| 第四句 | 七(红)、十(绿)、土(黑) | 七十土 |

| 第六句 | 八(红)、九(绿)、子(黑) | 八九子 |
| 第七句 | 佳(红)、作(绿)、亡(黑) | 佳作亡 |
| 第八句 | 福(红)、禄(绿)、寿(黑) | 福禄寿 |

### 2.2 字牌属性
- **总牌数**: 24种 × 4张 = 96张
- **特殊字牌**: 上、福（基础4胡）
- **颜色规则**: 每句第一个字红色，中间字绿色，末尾字黑色
- **每句为一句子组合**，可用于判定顺子/句子

## 三、胡数计算规则

### 3.1 基础胡数
| 类型 | 胡数 | 说明 |
|------|------|------|
| 上/福 | 4胡 | 手牌中持有上或福字牌的基础胡数 |
| 对子（上上/福福） | 8胡 | 两张相同的上或福 |
| 对子（其它） | 0胡 | 两张相同的普通字 |
| 半靠（含上/福） | 4胡 | 两张属于同一句话且含上或福 |
| 半靠（其它） | 0胡 | 两张属于同一句话的普通字 |
| 手牌一砍（普通） | 3胡 | 手牌中组成一砍（三张相同普通字） |
| 手牌一砍（上/福） | 12胡 | 手牌中组成一砍（三张相同上或福） |
| 碰除一砍（普通） | 2胡 | 碰牌后组成的一砍（普通字） |
| 碰除一砍（上/福） | 12胡 | 碰牌后组成的一砍（上或福） |
\| 招（普通） | 6胡 | 招牌（四张相同普通字） |
| 招（上/福） | 16胡 | 招牌（四张相同上或福） |

### 3.2 牌型状态定义

#### 3.2.1 对子
- **定义**: 两张相同的字牌
- **示例**: 上上、大大、福福
- **胡数**: 
  - 上上、福福：8胡
  - 其它对子：0胡

#### 3.2.2 半靠
- **定义**: 两张属于同一句话中的两个字
- **示例**: 
  - 上大人句：上大、大人、上人
  - 丘乙己句：丘乙、乙己、丘己
  - 化三千句：化三、三千、化千
  - 七十土句：七十、十土、七土
  - 尔小生句：尔小、小生、尔生
  - 八九子句：八九、九子、八子
  - 佳作亡句：佳作、作亡、佳亡
  - 福禄寿句：福禄、禄寿、福寿
- **胡数**: 
  - 含上/福的半靠：4胡（如上大、大人、上人、福禄、禄寿、福寿）
  - 其他半靠：0胡
- **用途**: 用于判断清胡等特殊胡牌类型

#### 3.2.3 半靠判定代码
```typescript
class HalfPairDetector {
  // 判断两张牌是否为半靠
  isHalfPair(card1: Card, card2: Card): boolean {
    // 必须属于同一句话
    if (card1.sentence !== card2.sentence) return false;
    
    // 不能是相同的牌（相同则是对子）
    if (card1.character === card2.character) return false;
    
    // 同一句话中的两张不同牌即为半靠
    return true;
  }
  
  // 获取手牌中所有的半靠
  getHalfPairs(cards: Card[]): HalfPair[] {
    const halfPairs: HalfPair[] = [];
    
    for (let i = 0; i < cards.length; i++) {
      for (let j = i + 1; j < cards.length; j++) {
        if (this.isHalfPair(cards[i], cards[j])) {
          halfPairs.push({
            cards: [cards[i], cards[j]],
            sentence: cards[i].sentence,
            characters: [cards[i].character, cards[j].character]
          });
        }
      }
    }
    
    return halfPairs;
  }
  
  // 检查是否有特定句子的半靠
  hasSentenceHalfPair(cards: Card[], sentence: number): boolean {
    const sentenceCards = cards.filter(c => c.sentence === sentence);
    return sentenceCards.length === 2 && 
           sentenceCards[0].character !== sentenceCards[1].character;
  }
  
  // 检查是否有上大人的半靠
  hasShangDaRenHalfPair(cards: Card[]): boolean {
    return this.hasSentenceHalfPair(cards, 1);
  }
  
  // 检查是否有福禄寿的半靠
  hasFuLuShouHalfPair(cards: Card[]): boolean {
    return this.hasSentenceHalfPair(cards, 8);
  }
}

interface HalfPair {
  cards: Card[];
  sentence: number;
  characters: string[];
}
```

### 3.3 胡数累加规则
- 胡数可叠加计算
- 不同组合的胡数相加得出总胡数

### 3.3 胡数计算示例
```
手牌: 上上上 + 大大大 + 人人 + 福福福
计算: 
- 上砍(手牌): 12胡 (上/福砍，已包含基础胡)
- 大砍(手牌): 3胡 (普通砍)
- 人对: 0胡
- 福砍(手牌): 12胡 (上/福砍，已包含基础胡)
总胡数: 12 + 3 + 0 + 12 = 27胡 (台胡)
```

**注意**: 上/福的基础胡数已包含在上/福砍的胡数中，不重复计算。

## 四、胡牌类型定义

### 4.1 胡牌方式
胡牌分为两种方式：
- **点炮**: 胡别人打出的牌，使用基础倍数
- **自摸**: 自己摸到胡牌，在点炮倍数基础上+1倍

### 4.2 普通胡牌（按胡数区分）
| 类型 | 胡数要求 | 点炮倍数 | 自摸倍数 | 说明 |
|------|----------|----------|----------|------|
| 普通胡 | 12~21胡 | 0倍 | 1倍 | 标准胡牌 |
| 卡胡 | 11胡 | 1倍 | 2倍 | 最低胡牌门槛 |
| 台卡 | 22胡 | 2倍 | 3倍 | 台数+卡胡 |
| 台胡 | 23~32胡 | 1倍 | 2倍 | 达到台数要求 |
| 重台卡 | 33胡 | 7倍 | 8倍 | 重台+卡胡 |
| 重台胡 | ≥34胡 | 6倍 | 7倍 | 最高普通胡 |

### 4.3 特殊胡牌类型

#### 4.3.1 枯胡
- **条件**: 手牌全部是坎（三张相同）外加一对，没有吃过句子
- **特点**: 纯粹靠坎组合胡牌
- **点炮倍数**: 5倍
- **自摸倍数**: 6倍

#### 4.3.2 清枯胡
- **条件**: 枯胡 + 清一色（单一颜色）
- **特点**: 所有字牌颜色相同
- **点炮倍数**: 6倍
- **自摸倍数**: 7倍

#### 4.3.3 清枯台卡
- **条件**: 清枯 + 台卡（22胡）
- **特点**: 清枯与台卡的组合
- **点炮倍数**: 8倍
- **自摸倍数**: 9倍

#### 4.3.4 黑元
- **条件**: 
  - 手牌除了一个半靠，其它全是句子
  - 只有吃的牌（没有碰牌）
  - 手牌和吃的牌没有"上大人"和"福禄寿"任何一个字
- **特点**: 排除特殊句子的纯句子组合
- **点炮倍数**: 4倍
- **自摸倍数**: 5倍

#### 4.3.5 红元
- **条件**:
  - 手牌除了一个半靠，其它全是句子
  - 只有吃的牌（没有碰牌）
  - 手牌和吃的牌必须保证上/福字数量≥3个
- **精数细分**:
  | 类型 | 上/福数量 | 点炮倍数 | 自摸倍数 |
  |------|-----------|----------|----------|
  | 红元3精 | 3个 | 3倍 | 4倍 |
  | 红元4精 | 4个 | 4倍 | 5倍 |
  | 红元5精 | 5个 | 5倍 | 6倍 |
  | 红元6精 | 6个 | 6倍 | 7倍 |
  | 红元7精 | 7个 | 7倍 | 8倍 |

#### 4.3.6 十对
- **条件**: 手牌由10个对子组成

## 五、游戏规则

### 5.1 游戏配置
- **玩家人数**: 3人
- **总牌数**: 96张（24种 × 4张）
- **每组局数**: 8局
- **发牌数量**: 
  - 庄家: 20张
  - 其他玩家: 各19张
- **剩余牌**: 96 - 20 - 19 - 19 = 38张作为牌堆

### 5.2 庄家规则
- **首局**: 按顺序轮流坐庄（玩家1 → 我 → 玩家2 → 玩家1...）
- **连庄条件**:
  - 庄家胡牌 → 连庄
  - 没有人胡牌（流局）→ 庄家连庄
- **换庄条件**: 非庄家胡牌 → 下一位玩家坐庄

### 5.3 游戏流程
1. **洗牌**: 随机打乱96张牌
2. **发牌**: 庄家发20张，其他玩家各发19张
3. **设置瓢分**: 从庄家开始，每位玩家设置瓢分
4. **摸牌**: 玩家轮流从牌堆摸牌
5. **打牌**: 玩家选择一张牌打出
6. **吃/碰/招**: 
   - 吃: 只能吃上家打出的牌组成句子
   - 碰: 任意玩家打出的牌可以与手牌组成砍
   - 招: 手牌已有三张相同，吃到第四张，**招完后自动从牌堆补齐一张牌**
7. **胡牌**: 达到胡牌条件即可胡牌
8. **结算**: 计算得分，更新总分

```typescript
interface CountdownConfig {
  duration: number;           // 总时长（毫秒）
  warningTime: number;        // 警告时间（毫秒）
  urgentTime: number;         // 紧急时间（毫秒）
  autoAction: () => void;     // 超时自动操作
}

class CountdownTimer {
  private timer: NodeJS.Timeout | null = null;
  private remainingTime: number = 0;
  private config: CountdownConfig;
  private onTick: (remaining: number) => void;
  
  constructor(config: CountdownConfig) {
    this.config = config;
  }
  
  // 开始倒计时
  start(): void {
    this.remainingTime = this.config.duration;
    this.timer = setInterval(() => {
      this.remainingTime -= 1000;
      
      // 触发tick回调
      if (this.onTick) {
        this.onTick(this.remainingTime);
      }
      
      // 警告时间提示
      if (this.remainingTime === this.config.warningTime) {
        this.playWarningSound();
      }
      
      // 紧急时间提示
      if (this.remainingTime <= this.config.urgentTime) {
        this.playUrgentSound();
      }
      
      // 超时
      if (this.remainingTime <= 0) {
        this.stop();
        this.playTimeoutSound();
        this.config.autoAction();
      }
    }, 1000);
  }
  
  // 停止倒计时
  stop(): void {
    if (this.timer) {
      clearInterval(this.timer);
      this.timer = null;
    }
  }
  
  // 获取剩余时间
  getRemainingTime(): number {
    return this.remainingTime;
  }
  
  // 播放警告声音
  private playWarningSound(): void {
    soundManager.play(SoundType.ALERT);
  }
  
  // 播放紧急声音
  private playUrgentSound(): void {
    soundManager.play(SoundType.COUNTDOWN);
  }
  
  // 播放超时声音
  private playTimeoutSound(): void {
    soundManager.play(SoundType.ALERT);
  }
}

// 使用示例

  duration: 30000,           // 30秒
  warningTime: 10000,        // 10秒时警告
  urgentTime: 5000,          // 5秒时紧急

    // 自动摸牌出牌
    if (gameState === GameState.PLAYER_TURN) {
      gameController.autoDrawAndDiscard();
    } else if (gameState === GameState.WAITING_RESPONSE) {
      gameController.autoPass();
    }
  }
});

// 玩家回合开始时启动倒计时

// 玩家操作后停止倒计时
```

```typescript
    const zhaoCards: ZhaoCard[] = [];
    const counts = this.countCards(dealer.handCards);

          character: char,
          cards: dealer.handCards.filter(c => c.character === char)
        });
      }

    if (shouldZhao) {

      const cards = zhaoCard.cards;
      dealer.handCards = dealer.handCards.filter(c => !cards.includes(c));
      
      dealer.melds.push({

        cards,

      this.playZhaoAnimation(cards);
      this.soundManager.play(SoundType.ZHAO);
      
      // 自动补牌
      this.autoDrawCard(dealer);
    }
    // 如果不招，直接进入出牌阶段
  }
}
```

### 5.3.6 招牌判定逻辑

```typescript
class ZhaoDetector {
  // 检查是否可以招（别人出牌时）
  canZhaoOnDiscard(player: Player, discardedCard: Card): boolean {
    // 检查手牌中是否有3张相同的字
    const count = player.handCards.filter(c => c.character === discardedCard.character).length;
    return count === 3;
  }
  
  // 检查是否可以招（自摸时）
  canZhaoOnDraw(player: Player, drawnCard: Card): boolean {
    // 情况1：手牌中有3张相同的字
    const handCount = player.handCards.filter(c => c.character === drawnCard.character).length;
    if (handCount === 3) return true;
    
    // 情况2：已碰的牌中可以招
    const pengMeld = player.melds.find(
      m => m.type === MeldType.TRIPLET && 
           m.source === MeldSource.PENG && 
           m.cards[0].character === drawnCard.character
    );
    if (pengMeld) return true;
    
    return false;
  }
  
  // 执行招牌操作
  executeZhao(player: Player, card: Card, isFromDiscard: boolean): void {
    // 情况1：别人出牌招
    if (isFromDiscard) {
      // 从手牌中移除3张相同的牌
      const sameCards = player.handCards.filter(c => c.character === card.character).slice(0, 3);
      player.handCards = player.handCards.filter(c => !sameCards.includes(c));
      
      // 添加招到已组合
      player.melds.push({
        type: MeldType.QUARTET,
        cards: [...sameCards, card],
        source: MeldSource.ZHAO,
        huValue: 6
      });
    }
    // 情况2：自摸招
    else {
      // 检查是否是碰变招
      const pengMeld = player.melds.find(
        m => m.type === MeldType.TRIPLET && 
             m.source === MeldSource.PENG && 
             m.cards[0].character === card.character
      );
      
      if (pengMeld) {
        // 碰变招：更新已碰的组合
        pengMeld.type = MeldType.QUARTET;
        pengMeld.cards.push(card);
        pengMeld.huValue = 6;
      } else {
        // 手牌招：从手牌中移除3张相同的牌
        const sameCards = player.handCards.filter(c => c.character === card.character).slice(0, 3);
        player.handCards = player.handCards.filter(c => !sameCards.includes(c));
        
        // 添加招到已组合
        player.melds.push({
          type: MeldType.QUARTET,
          cards: [...sameCards, card],
          source: MeldSource.ZHAO,
          huValue: 6
        });
      }
    }
    
    // 自动补牌
    this.autoDrawCard(player);
  }
  
  // 自动补牌
  private autoDrawCard(player: Player): void {
    // 从牌堆摸一张牌
    const card = this.deck.draw();
    if (card) {
      player.handCards.push(card);
      // 播放摸牌动画和声音
      this.playDrawAnimation(card);
      this.soundManager.play(SoundType.DRAW);
    }
  }
}
```

### 5.4 操作优先级规则

#### 5.4.1 操作优先级顺序
从高到低：**胡 > 招 > 碰 > 吃**

#### 5.4.2 基本规则
**重要规则：自己打出的牌不能自己吃/碰/招/胡/过**

- 打出牌的玩家不能对自己打出的牌进行任何操作
- 只有其他玩家才能对打出的牌进行吃/碰/招/胡操作
- 这条规则适用于所有玩家（包括AI和人类玩家）

#### 5.4.3 玩家1出牌场景详解
当玩家1打出一张牌时，按以下流程处理：

```
玩家1出牌
    ↓
检查玩家2是否可以胡/招/碰？
    ├─ 是 → 玩家2优先操作
    │       ├─ 玩家2选择胡 → 游戏结束，结算
    │       ├─ 玩家2选择招 → 玩家2招牌成功，轮到玩家2摸牌打牌
    │       ├─ 玩家2选择碰 → 玩家2碰牌成功，轮到玩家2摸牌打牌
    │       └─ 玩家2选择过 → 进入下一步
    │
    └─ 否 → 直接进入下一步
    ↓
检查我是否可以胡/招/碰/吃？
    ├─ 可以胡 → 显示[胡牌]按钮
    ├─ 可以招 → 显示[招牌]按钮
    ├─ 可以碰 → 显示[碰牌]按钮
    ├─ 可以吃 → 显示[吃牌]按钮（只能吃上家的牌）
    │
    └─ 如果玩家2不能胡/招/碰：
        ├─ 不显示[过]按钮
        └─ 只显示其他可操作按钮
    ↓
我选择操作
    ├─ 胡牌 → 游戏结束，结算
    ├─ 招牌 → 我招牌成功，轮到我摸牌打牌
    ├─ 碰牌 → 我碰牌成功，轮到我摸牌打牌
    ├─ 吃牌 → 我吃牌成功，轮到我打牌
    └─ 过（如果显示）→ 轮到我摸牌
```

#### 5.4.3 各玩家出牌场景汇总

**响应顺序规则**：按操作优先级排序，而非座位顺序
- 优先级：胡 > 招 > 碰 > 吃
- 同优先级按座位顺序

| 出牌者 | 下家（可吃） | 响应检查顺序 |
|--------|--------------|--------------|
| 玩家1 | 我 | 按优先级检查：先检查能胡/招/碰的玩家 → 再检查下家吃牌 |
| 我 | 玩家1 | 按优先级检查：先检查能胡/招/碰的玩家 → 再检查下家吃牌 |
| 玩家2 | 玩家1 | 按优先级检查：先检查能胡/招/碰的玩家 → 再检查下家吃牌 |

**玩家2出牌场景详解**：
```
玩家2出牌
    ↓
检查我和玩家1的操作优先级
    ↓
按优先级排序响应顺序
    ├─ 我可以胡/招/碰 → 我优先操作
    │   ├─ 显示: [胡牌] [招牌] [碰牌] [过]
    │   ├─ 我选择胡 → 游戏结束
    │   ├─ 我选择招/碰 → 我操作成功
    │   └─ 我选择过 → 玩家1可以操作
    │
    ├─ 玩家1可以胡/招/碰 → 玩家1操作
    │   ├─ 显示: [胡牌] [招牌] [碰牌] [过]
    │   └─ 玩家1选择操作
    │
    └─ 都不能胡/招/碰 → 检查玩家1吃牌
        ├─ 玩家1可以吃 → 显示: [吃牌]
        └─ 玩家1不能吃 → 玩家1摸牌
```

#### 5.4.4 操作按钮显示规则

**核心原则**：
1. 如果我有更高优先级的操作，显示所有可能操作 + [过]
2. 如果我选择[过]，下一个玩家才能操作
3. 如果没有更高优先级的玩家，不显示[过]按钮

**场景1：玩家2出牌，我可以胡，玩家1可以碰**

**场景2：玩家2出牌，我可以碰，玩家1也可以碰**

```
我选择"碰牌" → 我碰牌成功
我选择"过" → 玩家1操作
玩家1界面显示：[碰牌] [过]
```

**场景3：玩家2出牌，只有我可以吃（我是下家）**
```

（不显示[过]，因为没有更高优先级的玩家）
```

**场景4：玩家1出牌，玩家2可以碰，我可以吃**
```

玩家2界面显示：[碰牌] [过]

玩家2选择"过" → 我操作
```

**场景5：玩家1出牌，玩家2不能操作，我可以吃**

```
（不显示[过]）
```

**场景6：玩家2出牌，我可以胡/招/碰**
```
我有最高优先级操作
我的界面显示：[胡牌] [招牌] [碰牌] [过]

我选择任意操作 → 执行对应操作
我选择"过" → 玩家1操作（如果玩家1有操作）
```

#### 5.4.5 吃牌限制
- **只能吃上家的牌**：
  - 玩家1的上家是玩家2
  - 玩家2的上家是我
  - 我的上家是玩家1
- 吃牌后必须打出一张牌

### 5.5 牌型组合
- **句子**: 同一句的三个字（如：上大人、丘乙己）
- **砍**: 三张相同的字（如：上上上）
- **招**: 四张相同的字（如：上上上上）
- **对**: 两张相同的字（如：上上）

### 5.6 吃牌规则详解
- **只能吃上家的牌**: 当上家打出一张牌时，如果自己手中有两张牌可以与该牌组成完整句子，则可以吃牌
- **吃牌示例**:
  - 上家打出"大"，自己手中有"上"和"人"，可以吃成"上大人"
  - 上家打出"乙"，自己手中有"丘"和"己"，可以吃成"丘乙己"

## 六、听牌状态

### 6.1 听牌定义
**听牌**是指玩家手牌已经组成有效的牌型，只差一张牌就可以胡牌的状态。

### 6.2 听牌条件分类

#### 6.2.1 十对听牌
- **条件**: 
  - 手上除了一张单牌，全部都是对子
  - 招牌算两个对子（4张相同的字算两对）
  - 不能有吃、碰、招操作
- **示例1（普通对子）**: 
  - 手牌：上上 + 大大 + 人人 + 丘丘 + 乙乙 + 己己 + 化化 + 三三 + 千千 + 七（单张）
  - 听牌：七（需要再来一张"七"组成第10对）
- **示例2（含4张相同）**: 
  - 手牌：上上上上 + 大大 + 人人 + 丘丘 + 乙乙 + 己己 + 化化 + 三 + 千（单张）
  - 说明：上上上上算两对，共9对+1单张
  - 听牌：三或千（需要再来一张组成第10对）

#### 6.2.2 黑元听牌
- **条件**: 通过再补一张牌符合黑元条件
- **黑元条件回顾**:
  - 手牌除了一个半靠，其它全是句子
  - 只有吃的牌（没有碰牌）
  - 手牌和吃的牌没有"上大人"和"福禄寿"任何一个字
- **示例**:
  - 已吃：丘乙己、化三千、七十土
  - 手牌：尔小 + 八九子 + 佳作亡
  - 听牌：生（补一张"生"组成尔小生，完成黑元）

#### 6.2.3 红元听牌
- **条件**: 通过再补一张牌符合红元条件
- **红元条件回顾**:
  - 手牌除了一个半靠，其它全是句子
  - 只有吃的牌（没有碰牌）
  - 手牌和吃的牌必须保证上/福字数量≥3个
- **示例**:
  - 已吃：上大人、丘乙己、化三千
  - 手牌：七十 + 尔小生 + 八九子 + 佳
  - 听牌：土（补一张"土"组成七十土，完成红元3精）

#### 6.2.4 枯胡听牌
- **条件**: 通过再补一张牌符合枯胡条件
- **枯胡条件回顾**:
  - 手牌全部是坎（三张相同）外加一对
  - 没有吃过句子
- **示例**:
  - 手牌：上上上 + 大大大 + 人人 + 福福福
  - 听牌：人（补一张"人"组成人砍，完成枯胡）

#### 6.2.5 其它听牌
- **条件**: 
  - 胡数必须>=11胡
  - 手牌没有4张相同的牌
  - 手牌排除所有句子和坎
  - 只有两个半靠或者一张单字
- **包括**: 卡胡、普通胡、台卡、台胡、重台卡、重台胡等
- **示例1（一张单字）**: 
  - 手牌：上大人 + 丘乙己 + 化三千 + 七十土 + 尔小生 + 八九子 + 佳作亡 + 福（单张）
  - 听牌：福（补一张"福"组成对子，达到17胡，普通胡）
- **示例2（两个半靠）**: 
  - 手牌：上大人 + 丘乙己 + 化三千 + 七十土 + 尔小生 + 八九子 + 佳作亡 + 大人（半靠）
  - 听牌：上（补一张"上"组成上大人句子）
  - 说明：两个半靠需要补一张牌组成完整句子

### 6.3 听牌类型优先级
听牌类型的判定优先级从高到低：
1. **十对听牌** - 最高优先级
2. **黑元听牌**
3. **红元听牌**
4. **枯胡听牌**
5. **其它听牌** - 最低优先级

### 6.4 听牌检测算法

```typescript
enum TingType {
  TEN_PAIRS = 'ten_pairs',     // 十对听牌
  HEI_YUAN = 'hei_yuan',       // 黑元听牌
  HONG_YUAN = 'hong_yuan',     // 红元听牌
  KU_HU = 'ku_hu',             // 枯胡听牌
  OTHER = 'other',             // 其它听牌
}

interface TingStatus {
  isTing: boolean;              // 是否听牌
  tingType: TingType;           // 听牌类型
  tingCards: Card[];            // 听哪些牌
  huCount: number;              // 当前胡数
  huType: HuType;               // 胡牌类型
}

class TingDetector {
  // 检测是否听牌
  detectTing(player: Player): TingStatus {
    // 按优先级检测听牌类型
    const checks = [
      () => this.checkTenPairsTing(player),
      () => this.checkHeiYuanTing(player),
      () => this.checkHongYuanTing(player),
      () => this.checkKuHuTing(player),
      () => this.checkOtherTing(player),
    ];
    
    for (const check of checks) {
      const result = check();
      if (result.isTing) {
        return result;
      }
    }
    
    return {
      isTing: false,
      tingType: null,
      tingCards: [],
      huCount: 0,
      huType: null,
    };
  }
  
  // 检测十对听牌
  private checkTenPairsTing(player: Player): TingStatus {
    // 条件：不能有吃碰招操作
    if (player.melds.length > 0) {
      return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
    }
    
    const counts = this.countCards(player.handCards);
    const countValues = Object.values(counts);
    
    // 检查是否是9对+1单张
    let pairCount = 0;
    let singleCard: string = null;
    
    for (const [char, count] of Object.entries(counts)) {
      if (count === 2) pairCount++;
      else if (count === 1) {
        if (singleCard) return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
        singleCard = char;
      }
      else if (count === 4) {
        // 招牌算两个对子
        pairCount += 2;
      }
      else {
        return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
      }
    }
    
    // 必须是9对+1单张
    if (pairCount === 9 && singleCard) {
      const tingCard = player.handCards.find(c => c.character === singleCard);
      return {
        isTing: true,
        tingType: TingType.TEN_PAIRS,
        tingCards: [tingCard],
        huCount: this.huCalculator.calculateTotalHu(player.handCards, player.melds),
        huType: SpecialHuType.TEN_PAIRS,
      };
    }
    
    return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
  }
  
  // 检测黑元听牌
  private checkHeiYuanTing(player: Player): TingStatus {
    // 条件：只有吃的牌，没有碰牌
    const hasPengOrZhao = player.melds.some(
      m => m.source === MeldSource.PENG || m.source === MeldSource.ZHAO
    );
    if (hasPengOrZhao) {
      return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
    }
    
    // 检查是否差一张牌组成完整句子组合
    const result = this.checkSentenceTing(player);
    if (!result.isTing) {
      return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
    }
    
    // 检查是否符合黑元条件（没有上大人和福禄寿）
    const allCards = [...player.handCards, ...player.melds.flatMap(m => m.cards)];
    const forbiddenChars = ['上', '大', '人', '福', '禄', '寿'];
    const hasForbidden = allCards.some(c => forbiddenChars.includes(c.character));
    
    if (!hasForbidden) {
      return {
        isTing: true,
        tingType: TingType.HEI_YUAN,
        tingCards: result.tingCards,
        huCount: this.huCalculator.calculateTotalHu(player.handCards, player.melds),
        huType: SpecialHuType.HEI_YUAN,
      };
    }
    
    return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
  }
  
  // 检测红元听牌
  private checkHongYuanTing(player: Player): TingStatus {
    // 条件：只有吃的牌，没有碰牌
    const hasPengOrZhao = player.melds.some(
      m => m.source === MeldSource.PENG || m.source === MeldSource.ZHAO
    );
    if (hasPengOrZhao) {
      return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
    }
    
    // 检查是否差一张牌组成完整句子组合
    const result = this.checkSentenceTing(player);
    if (!result.isTing) {
      return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
    }
    
    // 检查上/福数量
    const allCards = [...player.handCards, ...player.melds.flatMap(m => m.cards)];
    const shangFuCount = allCards.filter(
      c => c.character === '上' || c.character === '福'
    ).length;
    
    // 补牌后上/福数量需要>=3
    const tingCard = result.tingCards[0];
    const isShangOrFu = tingCard.character === '上' || tingCard.character === '福';
    const finalCount = shangFuCount + (isShangOrFu ? 1 : 0);
    
    if (finalCount >= 3) {
      // 确定红元精数
      let huType: SpecialHuType;
      if (finalCount === 3) huType = SpecialHuType.HONG_YUAN_3;
      else if (finalCount === 4) huType = SpecialHuType.HONG_YUAN_4;
      else if (finalCount === 5) huType = SpecialHuType.HONG_YUAN_5;
      else if (finalCount === 6) huType = SpecialHuType.HONG_YUAN_6;
      else huType = SpecialHuType.HONG_YUAN_7;
      
      return {
        isTing: true,
        tingType: TingType.HONG_YUAN,
        tingCards: result.tingCards,
        huCount: this.huCalculator.calculateTotalHu(player.handCards, player.melds),
        huType,
      };
    }
    
    return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
  }
  
  // 检测枯胡听牌
  private checkKuHuTing(player: Player): TingStatus {
    // 条件：没有吃过句子
    const hasChiSequence = player.melds.some(
      m => m.type === MeldType.SEQUENCE && m.source === MeldSource.CHI
    );
    if (hasChiSequence) {
      return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
    }
    
    const counts = this.countCards(player.handCards);
    const countValues = Object.values(counts);
    
    // 检查是否是若干坎+一对+单张
    let kanCount = 0;
    let pairCount = 0;
    let singleCard: string = null;
    
    for (const [char, count] of Object.entries(counts)) {
      if (count === 3) kanCount++;
      else if (count === 2) {
        if (pairCount > 0) return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
        pairCount++;
      }
      else if (count === 1) {
        if (singleCard) return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
        singleCard = char;
      }
      else {
        return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
      }
    }
    
    // 必须有至少1个坎，1个对子，1个单张
    if (kanCount >= 1 && pairCount === 1 && singleCard) {
      // 检查单张是否可以组成对子
      const tingCard = player.handCards.find(c => c.character === singleCard);
      
      // 检查胡数是否达到门槛
      const tempHand = [...player.handCards, tingCard];
      const huCount = this.huCalculator.calculateTotalHu(tempHand, player.melds);
      
      if (huCount >= 11) {
        return {
          isTing: true,
          tingType: TingType.KU_HU,
          tingCards: [tingCard],
          huCount,
          huType: SpecialHuType.KU_HU,
        };
      }
    }
    
    return { isTing: false, tingType: null, tingCards: [], huCount: 0, huType: null };
  }
  
  // 检测其它听牌
  private checkOtherTing(player: Player): TingStatus {
    const tingCards: Card[] = [];
    
    // 遍历所有可能的牌
    for (const cardDef of CARD_DEFINITIONS) {
      const tempCard = this.createCard(cardDef);
      const tempHand = [...player.handCards, tempCard];
      
      // 检查是否可以组成有效牌型
      if (this.canFormValidHand(tempHand, player.melds)) {
        const huCount = this.huCalculator.calculateTotalHu(tempHand, player.melds);
        
        // 检查胡数是否达到门槛
        if (huCount >= 11) {
          tingCards.push(tempCard);
        }
      }
    }
    
    if (tingCards.length > 0) {
      // 确定胡牌类型
      const huCount = this.huCalculator.calculateTotalHu(player.handCards, player.melds);

      return {
```

#### 6.6.1 AI听牌策略
```typescript
class AITingStrategy {
  // 决定是否进入听牌状态
  shouldEnterTing(player: AIPlayer): boolean {
    const tingStatus = this.tingDetector.detectTing(player);
    
    if (!tingStatus.isTing) return false;
    
    // 评估听牌价值
    const value = this.evaluateTingValue(tingStatus);
    
    // 根据听牌价值决定是否听牌
    return value > this.threshold;
  }
  
  // 评估听牌价值
  private evaluateTingValue(tingStatus: TingStatus): number {
    let value = 0;
    
    // 1. 听牌数量（听得越多越好）
    value += tingStatus.tingCards.length * 10;
    
    // 2. 胡数（胡数越高越好）
    value += tingStatus.huCount;
    
    // 3. 胡牌类型倍数
    value += this.getHuTypeMultiplier(tingStatus.huType) * 5;
    
    // 4. 剩余牌数（剩余越多越好）
    value += this.getRemainingCardCount(tingStatus.tingCards) * 2;
    
    return value;
  }
  
  // 获取剩余牌数
  private getRemainingCardCount(tingCards: Card[]): number {
    let count = 0;
    for (const card of tingCards) {
      // 计算牌堆中剩余的该牌数量
      count += this.getRemainingCount(card.character);
    }
    return count;
  }
}
```

### 6.6 支付计算公式
```
输家支付分数 = 底分 + 翻倍基数分 × 倍数 + 自己的瓢分

其中：
- 点炮：倍数 = 点炮倍数
- 自摸：倍数 = 自摸倍数（点炮倍数 + 1）
```

### 6.5 点炮计分示例
```
假设：底分=10分，翻倍基数分=5分
玩家A（赢家）点炮胡牌类型：卡胡（点炮1倍），瓢分=10分
玩家B（点炮者）瓢分=5分
玩家C（输家）瓢分=0分

玩家B支付 = 10 + 5 × 1 + 5 = 20分
玩家C支付 = 10 + 5 × 1 + 0 = 15分
玩家A得分 = 20 + 15 = 35分
```

### 6.6 自摸计分示例
```
假设：底分=10分，翻倍基数分=5分
玩家A（赢家）自摸胡牌类型：卡胡（自摸2倍），瓢分=10分
玩家B（输家）瓢分=5分
玩家C（输家）瓢分=0分

玩家B支付 = 10 + 5 × 2 + 5 = 25分
玩家C支付 = 10 + 5 × 2 + 0 = 20分
玩家A得分 = 25 + 20 = 45分
```

### 6.7 特殊胡牌计分示例
```
假设：底分=10分，翻倍基数分=5分
玩家A（赢家）自摸胡牌类型：红元4精（自摸5倍），瓢分=20分
玩家B（输家）瓢分=10分
玩家C（输家）瓢分=5分

玩家B支付 = 10 + 5 × 5 + 10 = 45分
玩家C支付 = 10 + 5 × 5 + 5 = 40分
玩家A得分 = 45 + 40 = 85分
```

### 6.8 点炮与自摸对比示例
```
假设：底分=10分，翻倍基数分=5分，胡牌类型：枯胡

点炮情况（枯胡点炮5倍）：
输家支付 = 10 + 5 × 5 + 瓢分 = 35 + 瓢分

自摸情况（枯胡自摸6倍）：
输家支付 = 10 + 5 × 6 + 瓢分 = 40 + 瓢分

自摸比点炮多：5分（即翻倍基数分 × 1）
```

```typescript
class ScoringCalculator {
  // 获取倍数（根据胡牌方式和类型）
  getMultiplier(huResult: HuResult): number {
    if (huResult.method === HuMethod.ZI_MO) {
      return huResult.ziMoMultiplier;

    return huResult.dianPaoMultiplier;

    return method === HuMethod.ZI_MO ? entry.ziMo : entry.dianPao;

  calculatePayment(
    loser: Player,
    settings: GameSettings,
    multiplier: number
  ): number {
    return settings.baseScore + settings.multiplierBase * multiplier + loser.piaoScore;
  }
  
  // 计算赢家得分
  calculateWinnerScore(
    winner: Player,
    losers: Player[],
    settings: GameSettings,
    huResult: HuResult
  ): number {
    const multiplier = this.getMultiplier(huResult);
    return losers.reduce((sum, loser) => {
      return sum + this.calculatePayment(loser, settings, multiplier);
    }, 0);
  }
  
  // 结算本局
  settleRound(
    winner: Player,
    losers: Player[],
    settings: GameSettings,
    huResult: HuResult
  ): RoundSettlement {
    const multiplier = this.getMultiplier(huResult);
    
    const payments = losers.map(loser => ({
      playerId: loser.id,
      payment: this.calculatePayment(loser, settings, multiplier)
    }));
    
    const winnerScore = payments.reduce((sum, p) => sum + p.payment, 0);
    
    return {
      winnerId: winner.id,
      huType: huResult.normalType || huResult.specialType || huResult.comboType,
      huMethod: huResult.method,
      huCount: huResult.huCount,
      multiplier,
      payments,
      winnerScore
    };
  }
}
```

```typescript
class HuCalculator {
  // 计算基础胡数（上/福）
  calculateBaseHu(cards: Card[]): number {
    let hu = 0;
    const hasShang = cards.some(c => c.character === '上');
    const hasFu = cards.some(c => c.character === '福');
    if (hasShang) hu += 4;
    if (hasFu) hu += 4;
    return hu;
  }
  
  // 计算一砍胡数
  calculateKanHu(melds: Meld[]): number {
    return melds
      .filter(m => m.type === MeldType.TRIPLET)
      .reduce((sum, m) => {
        if (m.source === MeldSource.HAND) return sum + 3;
        if (m.source === MeldSource.PENG) return sum + 2;
        return sum;
      }, 0);
  }
  
  // 计算招胡数
  calculateZhaoHu(melds: Meld[]): number {
    return melds
      .filter(m => m.type === MeldType.QUARTET)
      .length * 6;
  }
  
  // 总胡数计算
  calculateTotalHu(player: Player): number {
    const baseHu = this.calculateBaseHu(player.handCards);
    const kanHu = this.calculateKanHu(player.melds);
    const zhaoHu = this.calculateZhaoHu(player.melds);
    return baseHu + kanHu + zhaoHu;
  }
}
```

```typescript
class HuValidator {
  // 普通胡牌判定
  validateNormalHu(totalHu: number): NormalHuType | null {
    if (totalHu < 11) return null;
    if (totalHu === 11) return NormalHuType.KA_HU;
    if (totalHu >= 12 && totalHu <= 21) return NormalHuType.NORMAL_HU;
    if (totalHu === 22) return NormalHuType.TAI_KA_HU;
    if (totalHu >= 23 && totalHu <= 32) return NormalHuType.TAI_HU;
    if (totalHu === 33) return NormalHuType.ZHONG_TAI_KA_HU;
    return NormalHuType.ZHONG_TAI_HU;
  }
  
  // 枯胡判定
  validateKuHu(player: Player): boolean {
    const hasChiSequence = player.melds.some(
      m => m.type === MeldType.SEQUENCE && m.source === MeldSource.CHI
    );
    if (hasChiSequence) return false;
    
    const cardCounts = this.countCards(player.handCards);
    const counts = Object.values(cardCounts);
    
    const triplets = counts.filter(c => c === 3).length;
    const pairs = counts.filter(c => c === 2).length;
    
    return triplets >= 1 && pairs === 1;
  }
  
  // 清枯判定
  validateQingKu(player: Player): boolean {
    if (!this.validateKuHu(player)) return false;
    
    const allCards = [...player.handCards, ...player.melds.flatMap(m => m.cards)];
    const colors = new Set(allCards.map(c => c.color));
    
    return colors.size === 1;
  }
  
  // 黑元判定
  validateHeiYuan(player: Player): boolean {
    const hasPengOrZhao = player.melds.some(
      m => m.source === MeldSource.PENG || m.source === MeldSource.ZHAO
    );
    if (hasPengOrZhao) return false;
    
    const allCards = [...player.handCards, ...player.melds.flatMap(m => m.cards)];
    const forbiddenChars = ['上', '大', '人', '福', '禄', '寿'];
    const hasForbidden = allCards.some(c => forbiddenChars.includes(c.character));
    if (hasForbidden) return false;
    
    return this.checkSentencePattern(player);
  }
  
  // 红元判定
  validateHongYuan(player: Player): SpecialHuType | null {
    const hasPengOrZhao = player.melds.some(
      m => m.source === MeldSource.PENG || m.source === MeldSource.ZHAO
    );
    if (hasPengOrZhao) return null;
    
    if (!this.checkSentencePattern(player)) return null;
    
    const allCards = [...player.handCards, ...player.melds.flatMap(m => m.cards)];
    const shangFuCount = allCards.filter(
      c => c.character === '上' || c.character === '福'
    ).length;
    
    if (shangFuCount >= 5) return SpecialHuType.HONG_YUAN_5;
    if (shangFuCount === 4) return SpecialHuType.HONG_YUAN_4;
    if (shangFuCount === 3) return SpecialHuType.HONG_YUAN_3;
    
    return null;
  }
  
  // 10对判定
  validateTenPairs(player: Player): boolean {
    const cardCounts = this.countCards(player.handCards);
    const counts = Object.values(cardCounts);
    
    return counts.filter(c => c === 2).length === 10;
  }
  
  // 综合判定
  validateHu(player: Player, card?: Card): HuResult {
    const tempCards = card ? [...player.handCards, card] : player.handCards;
    const totalHu = this.calculateTotalHu(tempCards, player.melds);
    
    const result: HuResult = {
      canHu: false,
      multiplier: 0,
      huCount: totalHu,
```

1. 检查melds中是否有吃牌（句子）

### 9.3 红元/黑元检测算法
1. 检查是否只有吃的牌（句子组合）
2. 统计手牌和吃的牌中的特殊字
3. 红元: 检查上/福数量

### 9.4 清一色检测算法

2. 检查是否所有字牌颜色相同

### 9.5 计分算法
```

赢家得分 = 所有输家支付之和
```

### 11.3 响应优先级处理流程

```typescript
interface ResponseContext {
  discardedCard: Card;        // 打出的牌
  discarder: Player;          // 打牌者
  potentialResponders: Player[];  // 可能响应的玩家
  currentResponderIndex: number;  // 当前响应者索引
  responderOrder: ResponsePriority[];  // 响应顺序
}

interface ResponsePriority {
  player: Player;
  canHu: boolean;
  canZhao: boolean;
  canPeng: boolean;
  canChi: boolean;  // 只有下家才能吃
}

class ResponseHandler {
  // 计算响应优先级
  calculateResponsePriority(
    discardedCard: Card,
    discarder: Player,
    players: Player[]
  ): ResponsePriority[] {
    const order: ResponsePriority[] = [];
    
    // 按座位顺序检查所有玩家（除了打牌者）
    for (const player of players) {
      if (player.id === discarder.id) continue;
      
      const priority: ResponsePriority = {
        player,
        canHu: this.canHu(player, discardedCard),
        canZhao: this.canZhao(player, discardedCard),
        canPeng: this.canPeng(player, discardedCard),
        canChi: this.canChi(player, discardedCard, discarder)
      };
      
      // 如果有任何高优先级操作，加入队列
      if (priority.canHu || priority.canZhao || priority.canPeng) {
        order.push(priority);
      }
    }
    
    // 按优先级排序：胡 > 招 > 碰
    order.sort((a, b) => {
      if (a.canHu !== b.canHu) return a.canHu ? -1 : 1;
      if (a.canZhao !== b.canZhao) return a.canZhao ? -1 : 1;
      if (a.canPeng !== b.canPeng) return a.canPeng ? -1 : 1;
      return 0;
    });
    
    return order;
  }
  
  // 处理响应流程
  async handleResponse(context: ResponseContext): Promise<void> {
    const priorities = this.calculateResponsePriority(
      context.discardedCard,
      context.discarder,
      context.players
    );
    
    // 如果没有人可以胡/招/碰，检查下家是否可以吃
    if (priorities.length === 0) {
      const nextPlayer = this.getNextPlayer(context.discarder);
      if (this.canChi(nextPlayer, context.discardedCard, context.discarder)) {
        // 下家可以吃，显示吃牌选项
        await this.promptChi(nextPlayer, context.discardedCard);
      } else {
        // 没有人可以吃，下家摸牌
        this.transitionToDraw(nextPlayer);
      }
      return;
    }
    
    // 有高优先级玩家，按顺序处理
    for (const priority of priorities) {
      const response = await this.promptResponse(priority);
      
      if (response === ResponseType.HU) {
        this.handleHu(priority.player, context.discardedCard);
        return;
      }
      
      if (response === ResponseType.ZHAO) {
        this.handleZhao(priority.player, context.discardedCard);
        return;
      }
      
      if (response === ResponseType.PENG) {
        this.handlePeng(priority.player, context.discardedCard);
        return;
      }
      
      // 玩家选择过，继续下一个
    }
    
    // 所有高优先级玩家都过了，检查下家吃牌
    const nextPlayer = this.getNextPlayer(context.discarder);
    if (this.canChi(nextPlayer, context.discardedCard, context.discarder)) {
      await this.promptChi(nextPlayer, context.discardedCard);
    } else {
      this.transitionToDraw(nextPlayer);
    }
  }
  
  // 显示操作按钮
  getAvailableActions(priority: ResponsePriority, hasHigherPriorityPassed: boolean): string[] {
    const actions: string[] = [];
    
    if (priority.canHu) actions.push('胡牌');
    if (priority.canZhao) actions.push('招牌');
    if (priority.canPeng) actions.push('碰牌');
    if (priority.canChi) actions.push('吃牌');
    
    // 如果有更高优先级的玩家操作过，显示"过"按钮
    if (hasHigherPriorityPassed) {
      actions.push('过');
    }
    
    // 如果没有任何操作可选，显示"摸牌"
    if (actions.length === 0) {
      actions.push('摸牌');
    }
    
    return actions;
  }
}
```

```typescript
interface DiscardHistory {
  playerId: string;
  cards: Card[];
  timestamps: Date[];
}

class DiscardHistoryManager {
  private histories: Map<string, DiscardHistory> = new Map();
  
  // 添加出牌记录
  addDiscard(playerId: string, card: Card): void {
    const history = this.histories.get(playerId) || { 
      playerId, 
      cards: [], 
      timestamps: [] 
    };
    history.cards.push(card);
    history.timestamps.push(new Date());
    this.histories.set(playerId, history);
  }
  
  // 移除出牌记录（被吃/碰/招/胡时）
  removeDiscard(playerId: string, card: Card): boolean {
    const history = this.histories.get(playerId);
    if (!history) return false;
    
    const index = history.cards.findIndex(c => c.id === card.id);
    if (index === -1) return false;
    
    history.cards.splice(index, 1);
    history.timestamps.splice(index, 1);
    return true;
  }
  
  // 获取玩家出牌历史
  getHistory(playerId: string): Card[] {
    return this.histories.get(playerId)?.cards || [];
  }
  
  // 获取最后打出的牌
  getLastDiscard(playerId: string): Card | undefined {
    const history = this.histories.get(playerId);
    if (!history || history.cards.length === 0) return undefined;
    return history.cards[history.cards.length - 1];
  }
  
  // 清空所有历史
  clearAll(): void {
    this.histories.clear();
  }
}
```

```typescript
interface CardDisplay {
  character: string;      // 字符
  count: number;          // 数量（0表示没有）
  position: number;       // 句中位置（0=首字, 1=中字, 2=末字）
}

class HandCardRenderer {
  // 按句子组织手牌
  organizeBySentence(handCards: Card[]): Map<number, CardDisplay[]> {
    const sentenceMap = new Map<number, CardDisplay[]>();
    
    // 初始化8个句子
    for (let i = 1; i <= 8; i++) {
      sentenceMap.set(i, [
        { character: CARD_DEFINITIONS[(i-1)*3].character, count: 0, position: 0 },
        { character: CARD_DEFINITIONS[(i-1)*3+1].character, count: 0, position: 1 },
        { character: CARD_DEFINITIONS[(i-1)*3+2].character, count: 0, position: 2 },
      ]);
    }
    
    // 统计每个字的数量
    const counts = this.countCards(handCards);
    
    // 更新显示数据
    for (const [char, count] of Object.entries(counts)) {
      const cardDef = CARD_DEFINITIONS.find(d => d.character === char);
      if (cardDef) {
        const sentence = sentenceMap.get(cardDef.sentence);
        if (sentence) {
          sentence[cardDef.position].count = count;
        }
      }
    }
    
    return sentenceMap;
  }
  
  // 渲染手牌
  render(ctx: CanvasRenderingContext2D, handCards: Card[], startX: number, startY: number): void {
    const sentenceMap = this.organizeBySentence(handCards);
    const cardWidth = 60;
    const cardHeight = 80;
    const gapX = 10;
    const gapY = 5;
    
    let x = startX;
    
    // 从左到右渲染每个句子
    for (let sentence = 1; sentence <= 8; sentence++) {
      const cards = sentenceMap.get(sentence);
      if (!cards) continue;
      
      let y = startY;
      
      // 从上到下渲染每个位置
      for (const cardDisplay of cards) {
        if (cardDisplay.count > 0) {
          // 绘制牌面
          this.drawCard(ctx, x, y, cardDisplay.character);
          
          // 如果数量>1，绘制右上角数字
          if (cardDisplay.count > 1) {
            this.drawCountBadge(ctx, x + cardWidth - 15, y + 5, cardDisplay.count);
          }
        }
        
        y += cardHeight + gapY;
      }
      
      x += cardWidth + gapX;
    }
  }
  
  // 绘制牌面
  private drawCard(ctx: CanvasRenderingContext2D, x: number, y: number, character: string): void {
    // 绘制牌背景
    ctx.fillStyle = '#FFFFFF';
    ctx.fillRect(x, y, 60, 80);
    
    // 绘制字
    ctx.font = 'bold 24px Arial';
    ctx.fillStyle = '#000000';
    ctx.textAlign = 'center';
    ctx.textBaseline = 'middle';
    ctx.fillText(character, x + 30, y + 40);
  }
  
  // 绘制数量徽章
  private drawCountBadge(ctx: CanvasRenderingContext2D, x: number, y: number, count: number): void {
    // 绘制圆圈
    ctx.beginPath();
    ctx.arc(x, y, 12, 0, Math.PI * 2);
    ctx.fillStyle = '#FF0000';
    ctx.fill();
    
    // 绘制数字
    ctx.font = 'bold 14px Arial';
    ctx.fillStyle = '#FFFFFF';
    ctx.textAlign = 'center';
    ctx.textBaseline = 'middle';
    ctx.fillText(count.toString(), x, y);
  }
  
  // 统计字牌数量
  private countCards(cards: Card[]): Record<string, number> {
    return cards.reduce((acc, card) => {
      acc[card.character] = (acc[card.character] || 0) + 1;
      return acc;
    }, {} as Record<string, number>);
  }
}
```

```typescript
class DeckStackRenderer {
  private config: DeckStackConfig;
  private remainingCount: number = 96;
  
  constructor(config: DeckStackConfig = defaultDeckStackConfig) {
    this.config = config;
  }
  
  // 渲染牌堆层叠效果
  render(ctx: CanvasRenderingContext2D, x: number, y: number): void {
    const img = new Image();
    img.src = this.config.cardImage;
    
    // 从底层到顶层渲染
    for (let i = 0; i < this.config.stackCount; i++) {
      const offsetX = x + i * this.config.offsetX;
      const offsetY = y - i * this.config.offsetY;
      
      ctx.drawImage(img, offsetX, offsetY);
    }
    
    // 显示剩余张数
    if (this.config.showRemaining) {
      const textX = x + img.width / 2;
      const textY = this.config.remainingPosition === 'bottom' 
        ? y + img.height + 20 
        : y - 10;
      
      ctx.font = 'bold 16px Arial';
      ctx.fillStyle = '#FFD700';
      ctx.textAlign = 'center';
      ctx.fillText(`剩余: ${this.remainingCount}张`, textX, textY);
    }
  }
  
  // 更新剩余张数
  updateRemaining(count: number): void {
    this.remainingCount = count;
  }
  
  // 减少一张
  decreaseOne(): void {
    this.remainingCount = Math.max(0, this.remainingCount - 1);
  }
}
```

```typescript
class SoundManager {
  private sounds: Map<SoundType, HTMLAudioElement>;
  private playerVoices: Map<string, PlayerVoice>;
  private config: SoundConfig;
  
  constructor() {
    this.sounds = new Map();
    this.playerVoices = new Map();
    this.config = {
      enabled: true,
      volume: 0.5
    };
    this.loadSounds();
  }
  
  // 加载所有声音文件
  private loadSounds(): void {
    // 加载通用声音
    const commonSoundFiles: Record<SoundType, string> = {
      [SoundType.DEAL]: '/sounds/common/deal.mp3',
      [SoundType.SHUFFLE]: '/sounds/common/shuffle.mp3',
      [SoundType.WIN]: '/sounds/common/win.mp3',
      [SoundType.LOSE]: '/sounds/common/lose.mp3',
      [SoundType.BUTTON_CLICK]: '/sounds/common/button_click.mp3',
      [SoundType.CARD_CLICK]: '/sounds/common/card_click.mp3',
      [SoundType.ALERT]: '/sounds/common/alert.mp3',
      [SoundType.COUNTDOWN]: '/sounds/common/countdown.mp3',
    };
    
    for (const [type, path] of Object.entries(commonSoundFiles)) {
      const audio = new Audio(path);
      audio.volume = this.config.volume;
      this.sounds.set(type as SoundType, audio);
    }
    
    // 加载玩家声音角色
    this.loadPlayerVoices();
  }
  
  // 加载玩家声音角色
  private loadPlayerVoices(): void {
    const voiceConfigs = [
      { playerId: 'player1', voiceType: PlayerVoiceType.MALE },
      { playerId: 'me', voiceType: PlayerVoiceType.FEMALE },
      { playerId: 'player2', voiceType: PlayerVoiceType.CHILD },
    ];
    
    for (const config of voiceConfigs) {
      const voice = this.createPlayerVoice(config.playerId, config.voiceType);
      this.playerVoices.set(config.playerId, voice);
    }
  }
  
  // 创建玩家声音
  private createPlayerVoice(playerId: string, voiceType: PlayerVoiceType): PlayerVoice {
    const sounds = new Map<SoundType, HTMLAudioElement>();
    const basePath = `/sounds/voices/${voiceType}`;
    
    const playerSoundFiles: Record<SoundType, string> = {
      [SoundType.DISCARD]: `${basePath}/discard.mp3`,
      [SoundType.DRAW]: `${basePath}/draw.mp3`,
      [SoundType.CHI]: `${basePath}/chi.mp3`,
      [SoundType.PENG]: `${basePath}/peng.mp3`,
      [SoundType.ZHAO]: `${basePath}/zhao.mp3`,
      [SoundType.HU]: `${basePath}/hu.mp3`,
    };
    
    for (const [type, path] of Object.entries(playerSoundFiles)) {
      const audio = new Audio(path);
      audio.volume = this.config.volume;
      sounds.set(type as SoundType, audio);
    }
    
    return {
      playerId,
      voiceType,
      sounds,
    };
  }
  
  // 播放通用声音
  play(type: SoundType): void {
    if (!this.config.enabled) return;
    
    const sound = this.sounds.get(type);
    if (sound) {
      sound.currentTime = 0;
      sound.play().catch(() => {
        // 忽略自动播放限制错误
      });
    }
  }
  
  // 播放玩家声音（带角色）
  playPlayerSound(playerId: string, type: SoundType): void {
    if (!this.config.enabled) return;
    
    const voice = this.playerVoices.get(playerId);
    if (voice) {
      const sound = voice.sounds.get(type);
      if (sound) {
        sound.currentTime = 0;
        sound.play().catch(() => {
          // 忽略自动播放限制错误
        });
      }
    }
  }
  
  // 设置音量
  setVolume(volume: number): void {
    this.config.volume = Math.max(0, Math.min(1, volume));
    this.sounds.forEach(sound => {
      sound.volume = this.config.volume;
    });
    this.playerVoices.forEach(voice => {
      voice.sounds.forEach(sound => {
        sound.volume = this.config.volume;
      });
    });
  }
  
  // 开关声音
  toggle(enabled?: boolean): void {
    this.config.enabled = enabled ?? !this.config.enabled;
  }
}
```

#### 15.10.2 延时管理器
```typescript
class DelayManager {
  private config: DelayConfig;
  
  constructor(config: DelayConfig = defaultDelayConfig) {
    this.config = config;
  }
  
  // AI操作延时
  async waitForAIAction(): Promise<void> {
    await this.delay(this.config.aiActionDelay);
  }
  
  // AI思考延时
  async waitForAIThinking(): Promise<void> {
    await this.delay(this.config.aiThinkingDelay);
  }
  
  // 玩家操作延时
  async waitForPlayerAction(): Promise<void> {
    await this.delay(this.config.playerActionDelay);
  }
  
  // 动画延时
  async waitForAnimation(): Promise<void> {
    await this.delay(this.config.animationDelay);
  }
  
  // 消息提示延时
  async waitForMessage(): Promise<void> {
    await this.delay(this.config.messageDelay);
  }
  
  // 回合切换延时
  async waitForTurnSwitch(): Promise<void> {
    await this.delay(this.config.turnSwitchDelay);
  }
  
  // 通用延时
  private delay(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
  
  // 更新配置
  updateConfig(config: Partial<DelayConfig>): void {
    this.config = { ...this.config, ...config };
  }
}
```

|------|------|

| 砍 | 三张相同的字组成的刻子，如"上上上" |
| 招 | 四张相同的字组成的杠子，如"上上上上" |

| 胡数 | 衡量牌型价值的数值 |
| 胡牌 | 达到条件结束游戏 |
| 吃牌 | 吃上家打出的牌组成句子 |
| 碰牌 | 碰任意玩家打出的牌组成砍 |
| 招牌 | 手牌三张相同，吃第四张 |
| 底分 | 每局游戏的基础分数 |
| 瓢分 | 玩家自愿增加的额外分数 |

| 连庄 | 庄家继续坐庄 |
