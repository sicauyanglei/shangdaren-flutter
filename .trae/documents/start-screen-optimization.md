# 启动页面优化方案

## 当前问题
- 启动页面左右横排布局（Logo + 选项网格），纯文字选项，视觉单调
- 缺少氛围感和视觉层次

## 设计方案（5种）

### A. 居中竖排 + 字牌装饰（推荐）
- 竖排居中布局：Logo → 标题 → 选项面板 → 开始按钮
- 背景散落半透明字牌装饰（红/绿/黑三色）
- 选项面板毛玻璃效果（半透明背景 + backdrop-filter）
- 选项行内横排（标签+按钮组一行）

### B. 卡牌展示墙 + 浮动面板
- 背景铺满半透明字牌（网格排列）
- 中央浮动深色面板，金色标题 + 金色开始按钮
- Logo和标题横排显示

### C. 沉浸式牌桌 + 浮动字牌
- 模拟牌桌视角，底部桌面渐变
- 顶部浮动字牌动画（上下漂浮）
- 底部设置面板 + 开始按钮

### D. 古风卷轴 + 印章风格
- 深红古风配色，卷轴边框+角花装饰
- 红色印章Logo，中国风韵味
- 选中态红色高亮

### E. 现代卡片式 + 渐变光效
- 现代UI风格，浮动光晕背景
- 圆角卡片面板，按钮流光动画
- Logo徽章样式

## 实现步骤
1. 用户选择方案
2. 修改 `start_screen.dart` 的布局和样式
3. 如需装饰元素（字牌背景等），在Canvas或Widget中绘制
4. 编译release版本并推送到BlueStacks测试
