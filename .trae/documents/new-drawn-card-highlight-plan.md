# 人类玩家新摸牌标识效果设计方案

## 当前手牌布局

- 手牌宽度 60px，高度 224px（竖向长条）
- 按 sentence（组号1-8）分组，从左到右排列
- 同 sentence 内按字分层，从上到下叠放，同字叠放竖向偏移 64px
- 不同 sentence 组之间水平间距 2px
- 选中牌上移 28px
- 听牌徽章：右上角红色小圆点（半径6）

## 新摸牌标识方案

| 方案 | 名称 | 描述 |
|------|------|------|
| A | 顶部脉冲光带 | 卡牌顶部显示一条水平脉冲发光条（橙色），呼吸式闪烁 |
| B | 左侧竖向发光条 | 卡牌左侧显示一条竖向发光条（绿色），静态+微弱脉冲 |
| C | 右上角"新"标签 | 卡牌右上角显示红色"新"字标签，类似通知角标 |
| D | 背景半透明色块 | 卡牌底部添加半透明高亮色块背景，不改变卡牌本身 |
| E | 边框发光脉冲 | 卡牌整体边框发光脉冲（蓝色），1.5s周期 |
| F | 微幅上浮 | 卡牌微微上浮8-12px + 阴影加深，悬浮效果 |
| G | 入场滑入动画 | 新牌从右侧滑入到手牌位置（一次性动画） |
| H | 底部光晕 | 卡牌底部显示向上扩散的光晕，类似地面反光 |

## 实现步骤

1. 创建 HTML 文件，模拟人类玩家手牌布局（竖向长条卡牌，按组排列，同字叠放）
2. 展示 8 种方案，每种在最后一张手牌上应用效果
3. 在浏览器中预览
