# 出牌醒目标识效果设计方案

## 当前效果

当前出牌（弃牌区最后一张）的标识效果：
1. **正弦脉冲缩放**（1.0~1.1倍，周期1.5秒）
2. **金色边框**（线宽2~4px脉冲）
3. **双层金色发光**（blur半径8px和16px）

问题：金色发光在小尺寸弃牌卡牌（28×48px）上不够醒目，容易与背景混淆。

## 设计方案

创建一个 HTML 页面，展示以下 8 种不同的出牌醒目标识效果，每种效果附带动画演示：

| 方案 | 名称 | 描述 |
|------|------|------|
| A | 红色脉冲边框 | 红色边框 + 红色外发光，脉冲动画，红色在深色背景上更醒目 |
| B | 橙色呼吸光晕 | 橙色宽边框 + 多层扩散光晕，呼吸式动画 |
| C | 闪烁白框 | 白色边框快速闪烁（0.6s周期），类似提示/警告效果 |
| D | 背景高亮色块 | 卡牌底部添加高亮色背景块（如红色半透明），静态+微弱脉冲 |
| E | 放大+阴影 | 卡牌放大1.15倍 + 强投影效果，突出立体感 |
| F | 箭头指示器 | 卡牌上方显示向下箭头动画，指向出牌 |
| G | 周围暗化聚焦 | 周围卡牌变暗，出牌卡牌保持原色+边框，聚焦效果 |
| H | 彩虹渐变边框 | 边框颜色随时间渐变流动，视觉吸引力最强 |

## 实现步骤

1. 创建 HTML 文件 `e:\AI-PRJ\shangdaren-game\flutter_app\assets\html\discard_card_effects.html`
2. 页面布局：2列4行网格，每个格子展示一种效果
3. 每个格子包含：
   - 效果名称
   - 模拟弃牌区（3-4张小卡牌，最后一张应用效果）
   - 简短说明
4. 使用纯 CSS 动画实现所有效果
5. 卡牌样式模拟实际游戏中的弃牌卡牌（28×48px，深色背景）
