# 出牌醒目标识效果设计方案（扩展版）

## 已有方案回顾

A-H 共8种方案已在 `discard_card_effects.html` 中展示。

## 新增方案

| 方案 | 名称 | 描述 |
|------|------|------|
| I | 浮动悬浮 | 卡牌微微上下浮动（3-4px），像悬浮在桌面上 |
| J | 旋转光弧边框 | conic-gradient 旋转光弧沿边框流动，科技感 |
| K | 波纹扩散 | 从卡牌中心向外扩散的圆环波纹，类似水波 |
| L | 霓虹灯管 | 霓虹灯管风格发光，颜色鲜艳+灯管质感 |
| M | 抖动提醒 | 卡牌轻微水平抖动，类似手机震动提醒 |
| N | 底部光柱 | 卡牌下方有光柱向上照射，舞台聚光灯效果 |
| O | 描边流动 | 边框沿卡牌轮廓方向流动的动画（蚂蚁线/行军蚁） |
| P | 脉冲圆点标记 | 卡牌右上角显示脉冲闪烁的红色圆点，类似通知标记 |
